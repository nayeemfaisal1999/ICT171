My name is Nayeem Faisal, and I am based in Perth, Western Australia. I have been doing photography since 2019, both personally and professionally. I am currently studying a Bachelor's degree in Cybersecurity and Forensics at Murdoch University.

Beyond my studies and photography, I serve as a Guild Councillor for the Murdoch Guild, representing students and advocating for their interests within the university community.

🏛️ Murdoch Guild Representative

For this cloud project, I have created a photography portfolio website hosted on a cloud-based virtual server using DigitalOcean. The site showcases my online portfolio, displaying my work as a freelance photographer and editor.

DigitalOcean Ubuntu 22.04 LTS Apache2 HTML/CSS JavaScript

The website includes a landing page with a photo gallery, an About Me section featuring my background and photography journey, and comprehensive licensing information. The gallery highlights different categories of my work, such as portrait, landscape, and street photography.

The server runs Ubuntu 22.04 LTS, and I installed Apache2 to serve static HTML and CSS files. I used basic command line tools for server setup and deployed the site to the /var/www/html directory. The firewall settings were configured to allow inbound HTTP (port 80) and restrict SSH access to my IP address only.

I chose DigitalOcean for its simplicity, reliability, and excellent documentation. The platform provides perfect scalability for hosting static websites and offers robust infrastructure ideal for learning cloud deployment fundamentals.

In the future, I plan to expand this site with a blog section featuring behind-the-scenes stories from my photography sessions and student life at Murdoch University. I also intend to implement visitor analytics using privacy-focused, open-source tools.
